# Light-Shield

**Light-Shield** is a Chrome browser extension that softens harsh screen brightness and protects your eyes while browsing. It provides a customizable dimming overlay, letting you comfortably read websites, documents, or novels without going fully dark.

## Installation

1. Clone or download this repository.
2. Open Chrome ‚Üí `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the extension folder.
5. Click the **Light-Shield** icon and enjoy eye-friendly browsing.

---

## Usage

1. Click the **Light-Shield** extension icon in Chrome.
2. Use the toggle to enable or disable the overlay.
3. Open **Advanced Settings** to adjust opacity and overlay color.
4. The overlay applies immediately on the current tab.
